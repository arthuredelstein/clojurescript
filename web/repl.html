Datatypes

Functions

Calling (<function> <args*>)
Defining named functions (defn <name> [<args*>] |constraints| <actions*>)
Anonymous function (fn |name| [<args*>] |constraints| <actions*>)
Anonymous inline function #(<action> |%,%2,%3, or %&|)
